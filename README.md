## SwiftUI with TCA 프로젝트의 목표
1. SwiftUI 로 View 를 작성하는 능력 키우기
2. TCA 를 통해 단방향 데이터 흐름 배우기
3. iOS 뿐만 아니라 iPadOS, macOS 에도 호환되는 프로젝트 만들기 (OS 에 따른 분기처리)
4. Combine 을 적용하여 비동기 로직 구현하는 능력 키우기
